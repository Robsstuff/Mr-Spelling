The Rule

PLURAL RULE 3
consonant + Y → consonant + IES

When a noun ends in a consonant followed by Y, change the Y to I and add -ES. The Y is replaced — it does not stay in the word.

Exception: if the letter before the Y is a vowel (a, e, i, o, u), just add -S instead — for example, toy → toys, monkey → monkeys, key → keys.

baby → babies pony → ponies city → cities story → stories puppy → puppies berry → berries
